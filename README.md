# 🏠 My HomeLab Infrastructure

Este repositorio contiene la configuración de mi laboratorio personal basado en **Docker** y **LXC**, gestionado como infraestructura como código.

## 🛠️ Stack Tecnológico
- **OS:** Ubuntu 24.04 LTS (VM)
- **Contenedores:** Docker & Docker Compose
- **Servicios:** MariaDB (Base de datos), Adminer (Gestión DB)

## 🚀 Cómo desplegar
1. Clonar el repositorio.
2. Ejecutar `docker-compose up -d`.
3. Acceder a Adminer en `http://localhost:8080`.
